# betao-fe-assignment

This is a small app which renders a periodic table. Each element is grouped by color depending on it's element type. It's also possible to invert the entire table by hitting the invert button.

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).

## Customize configuration

See [Vite Configuration Reference](https://vitejs.dev/config/).

## Project Setup

```sh
npm install
```
